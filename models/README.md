This folder stores saved model artifacts, outputs, and visualizations generated during the training and evaluation process.

## Contents

After running [notebooks/02_model_training_and_evaluation.ipynb](../notebooks/02_model_training_and_evaluation.ipynb), this directory will contain:

- **Trained model files** (`.pkl` format)
- **Model performance metrics** (`.json` format)
- **Evaluation plots and visualizations** (`.png` format)
- **Feature importance rankings**
- **Calibration curves and risk tier analyses**
